# วันที่ 1 ฉันทำอะไร

วันนี้มาเริ่มเรียนกับโปรแกรมเต่านั่นเองงงงงงงงงง ฝึกการใช้งานการเขียนโปรแกรมเบื้องต้น<br><br>
โจทย์ในวันนี้คือการให้เต่าวาดรูปอะไรก็ได้<br><br>
เลยตั้งโจทย์กับตัวเองว่า อยากให้เต่าวาดรูปสี่เหลี่ยม หลายๆขนาดแบบสุ่ม แล้วก็สุ่มตำแหน่งที่วาดขึ้นมาด้วย<br><br><br>

```
import turtle

tao = turtle.Pen()
tao.shape("turtle")
```
เริ่มจาก import เต่าเข้ามา แล้วเรียกใช้งานเต่า<br><br>
```
random.randrange(range1, range2)
```
สุ่มแกน X,Y แล้วก็ตำแหน่ง ด้วยคำสั่งนี้
